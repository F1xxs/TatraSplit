# Task 09 — Accessibility pass (keyboard + semantics)

## Goal
Improve keyboard usability and assistive technology support.

## Scope
- Audit dialogs/sheets/tabs/navigation controls.
- Ensure focus is trapped/restored in overlays.
- Add/adjust aria labels and semantic roles where needed.

## Implementation
- Fix keyboard navigation gaps.
- Improve screen-reader text for icon-only actions.
- Verify tab order and focus visibility across key flows.

## Acceptance criteria
- All main flows are keyboard-accessible.
- Dialog/sheet interactions are accessible and predictable.
- No critical accessibility regressions in navigation/forms.
